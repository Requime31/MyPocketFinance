# MyPocketFinance

MyPocketFinance is a modern personal finance app that helps you track expenses, manage income and save for your goals with a clean, focused UI.

## What it does

- **Dashboard** – shows total balance, income, expenses and spending breakdown by category.
- **Transactions** – lets you quickly add, edit and review income and expense records.
- **Savings goals** – create goals (vacation, emergency fund, gadgets, etc.) and track progress over time.
- **Analytics** – see reports with spending by category, trends and cashflow, plus aggregated stats for your goals.

The app is built with **Swift** and **SwiftUI** and is designed to feel native on iOS with smooth animations and a minimalist visual style.

## Requirements

- macOS with the latest stable **Xcode** installed.
- An **Apple ID** added in Xcode (required for running on a real device).
- iOS 16+ (recommended) on simulator or device.

## Install and run

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   cd MyPocketFinance
   ```
2. Open the project in Xcode:
   - Open `MyPocketFinance.xcodeproj`.
3. Select a target:
   - Choose an iOS Simulator, or
   - Connect your iPhone with a cable and trust the computer/device.
4. If running on a real device, configure signing:
   - In Xcode, select the `MyPocketFinance` target.
   - Open **Signing & Capabilities**.
   - Enable **Automatically manage signing**.
   - Select your **Team** (Apple ID account).
5. Build and run:
   - Press **Run** (`Cmd + R`).

## Run on your iPhone (first time)

- On the device, go to **Settings > Privacy & Security** and allow the developer profile if prompted.
- If iOS asks to trust the app/developer, confirm trust, then launch the app again.
